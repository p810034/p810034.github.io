github靜態架站紀錄

  1. 建立 Github Pages
    1.1 建立一個新的repository
    1.2 專案名稱為 username.github.io
    1.3 進入專案，到 Settings 頁面
    1.4 GitHub Pages，Choose a theme
    1.5 https://username.github.io/

  2. 建立Hexo環境
    2.1 先安裝Node.js、Git
    2.2 終端機安裝hexo
    2.2.1 npm install hexo-cli -g
    2.3 初始化 Hexo 部落格
    2.3.1 hexo init name //name的名稱自己取
    2.3.2 cd name
    2.3.3 npm install
    2.3.4 npm install hexo-deployer-git –save
    2.4 設定部落格
    2.4.1 編輯_config.yml
    (1)title: 標題 #部落格標題
    (2)subtitle: 副標題 #副標題
    (3)description: #網站描述
    (4)keywords: #網站關鍵字(以逗號隔開)，方便 SEO
    (5)author: 張三 #作者
    (6)language: zh-TW #語言
    (7)timezone: #使用系統時間留空
    (8)url: https://username.github.io/ #自己的url
    2.5 設定部屬
    2.5.1 編輯_config.yml
    (1)deploy:
    type: git
    repo: https://github.com/username/username.github.io.git
    branch: main #看你的branch是哪個

  3. 部屬到Github
    3.1 終端機輸入
    3.1.1 git config –global user.email “you@example.com“ //設定git user信箱，只有初次需要
    3.1.2 git config –global user.name “Your Name” //設定git user name，只有初次需要
    3.1.3 hexo cl //清除靜態檔案
    3.1.4 hexo g //產生靜態檔案
    3.1.5 hexo d //部署至 Github Pages
